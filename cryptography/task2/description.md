### **Хеширование паролей**

##### *Описание задачи*

В этой задаче мы напишем функцию на Python для вычисления хеша с использованием алгоритма SHA-256 для строки вида "пароль:соль".

##### *Цель*

Написать функцию на Python с именем hash_password, которая принимает на вход пароль (строку) и соль (строку), объединяет их, вычисляет хеш с использованием алгоритма SHA-256 и возвращает результат в виде шестнадцатеричной строки.

Функция должна:

* Принимать два аргумента: password (строка) и salt (строка).
* Объединять пароль и соль в одну строку (сначала пароль, потом соль).
* Преобразовывать объединенную строку в байты (используя кодировку UTF-8).
* Вычислять хеш SHA-256 от этих байтов.
* Возвращать полученный хеш в виде строки шестнадцатеричных символов (например, 'a1b2c3d4...').
