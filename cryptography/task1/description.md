### **Хеширование (MD5 и SHA-256)**

##### *Описание задачи*

В этой задаче мы вычислим два типа хешей: MD5 и SHA-256 для заданной строки.

##### *Цель*

Написать программу на Python, которая:

1. Прочитать одну строку текста со стандартного ввода (input()).
2. Вычислить для этой строки MD5 хеш.
3. Вычислить для этой строки SHA-256 хеш.
4. Вывести сначала MD5 хеш, а затем SHA-256 хеш, каждый на новой строке.
