### **Найти строки, начинающиеся с даты (YYYY-MM-DD)**

##### *Описание задачи*

Многие форматы логов начинаются с метки времени, например, с даты в формате "Год-Месяц-День". Это позволяет легко фильтровать события по дате или сортировать их хронологически. Умение отбирать строки, начинающиеся с определенного формата даты, — важный навык при анализе логов.

Наша задача — написать программу, которая читает многострочный текст и выводит только те строки, которые начинаются с даты в формате ГГГГ-ММ-ДД.

##### *Цель*

Написать программу на Python, которая:

1. Читает многострочный текст лога из стандартного потока ввода (stdin).
2. Обрабатывает этот текст построчно.
3. Для каждой строки проверяет, начинается ли она с последовательности символов, соответствующей формату ГГГГ-ММ-ДД (четыре цифры, дефис, две цифры, дефис, две цифры).
4. Если строка начинается с указанного формата даты, программа должна вывести эту строку целиком, включая символ перевода строки, с которым она была прочитана.
5. Если строка не начинается с этого формата, ее выводить не нужно.
