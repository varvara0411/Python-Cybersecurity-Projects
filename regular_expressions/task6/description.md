### **Подсчет сообщений об отказе (Failed)**

##### *Описание задачи*

При анализе больших лог-файлов часто требуется не просто найти ошибки, а посчитать их количество или количество событий определенного типа. Например, сколько раз за период времени произошел сбой аутентификации или отказ в доступе?

В этой задаче мы будем использовать регулярные выражения для подсчета строк в многострочном логе, которые содержат конкретное слово-индикатор — "Failed".

##### *Цель*

Написать программу на Python, которая:

1. Читает многострочный текст лога из стандартного потока ввода (stdin).
2. Обрабатывает этот текст построчно.
3. Для каждой строки проверяет, содержит ли она слово "Failed". Проверка должна быть чувствительной к регистру (т.е. "failed" или "FAILED" не считаются).
4. Подсчитывает общее количество строк, содержащих слово "Failed".
5. Выводит на экран только итоговое количество найденных строк (в виде целого числа).
