### **Извлечение ID процесса (pid=число)**

##### *Описание задачи*

Системные логи часто содержат идентификаторы процессов (PID, Process ID). Это уникальные номера, которые операционная система присваивает каждому запущенному процессу. Анализ логов может потребовать извлечения этих PID для отслеживания действий конкретного процесса.

Часто PID указывается в логе в формате pid=ЧИСЛО, например, pid=12345.

Наша задача — использовать регулярные выражения, чтобы найти в строке лога фрагмент pid=ЧИСЛО и извлечь из него только само числовое значение PID.

##### *Цель*

Написать программу на Python, которая:

1. Получает на вход одну строку лога (через стандартный ввод).
2. Используя регулярные выражения, находит первое вхождение подстроки, начинающейся с "pid=", за которым следует последовательность из одной или более цифр.
3. Извлекает из найденного совпадения только ту часть, которая соответствует числовому значению PID (после "pid=").
4. Если совпадение найдено, выводит на экран только извлеченное число.
5. Если подстрока формата "pid=..." не найдена, программа не должна ничего выводить.
