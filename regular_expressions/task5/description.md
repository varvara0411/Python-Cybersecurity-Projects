### **Извлечение имени пользователя из строки "user=имя"**

##### *Описание задачи*

Логи аутентификации или аудита часто содержат информацию о том, какой пользователь выполнил определенное действие. Эта информация может быть представлена в виде пар "ключ=значение", например, user=jsmith или uid=1001.

Наша задача — использовать регулярные выражения, чтобы найти фрагмент user=имя_пользователя в строке лога и извлечь из него только само имя пользователя. Имя пользователя в данном случае может состоять только из букв латинского алфавита (a-z, A-Z), цифр (0-9), дефиса (-) и нижнего подчеркивания (_).

##### *Цель*

Написать программу на Python, которая:

1. Получает на вход одну строку лога (через стандартный ввод).
2. Используя регулярные выражения, находит первое вхождение подстроки, начинающейся с "user=", за которым следует последовательность из одного или более символов (буквы, цифры, дефисы и нижние подчеркивания.
3. Извлекает из найденного совпадения только ту часть, которая соответствует имени пользователя (после "user=").
4. Если совпадение найдено, выводит на экран только извлеченное имя пользователя.
5. Если подстрока формата "user=..." не найдена, программа не должна ничего выводить.
