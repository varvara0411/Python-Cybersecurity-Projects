### **Чтение значения из текстовой конфигурации (Key-Value)**

##### *Описание задачи*

Многие программы и скрипты используют простые текстовые файлы для хранения настроек. Часто используется формат "ключ-значение", где каждая строка содержит имя параметра (ключ) и его значение, разделенные, например, знаком равенства (=).

Представьте, что вы анализируете конфигурацию сетевого сервиса, и вам нужно узнать, какой порт он использует для прослушивания входящих соединений. Эта информация обычно хранится под ключом вроде `LISTEN_PORT`.

Ваша задача — прочитать многострочный текст конфигурации (поданный на стандартный ввод), найти строку с нужным ключом и извлечь его значение.

##### *Цель*

Написать программу на Python, которая:

1. Читает многострочный текст из стандартного потока ввода (stdin), представляющий собой конфигурацию.
2. Находит строку, которая начинается с ключа `LISTEN_PORT` (регистр важен).
3. Извлекает значение, идущее после знака = в этой строке. Учитывайте, что вокруг знака = могут быть пробелы, которые не являются частью значения.
4. Выводит на экран только найденное значение (без лишних пробелов по краям).
5. Если строка с ключом `LISTEN_PORT` не найдена, программа не должна ничего выводить.
