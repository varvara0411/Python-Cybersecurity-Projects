### **Парсинг XML. Конфигурация сервиса (имя)**

##### *Описание задачи*

XML (eXtensible Markup Language) — это язык разметки, похожий на HTML, но используемый для хранения и передачи структурированных данных. Он часто применяется в файлах конфигурации, протоколах обмена данными и форматах отчетов.

XML состоит из тегов (`<tag>...</tag>`), которые могут содержать текст, другие теги или иметь атрибуты (пары имя="значение" внутри открывающего тега).

Представьте, что у вас есть конфигурационный файл в формате XML, описывающий различные сервисы системы. Вам нужно узнать имя конкретного сервиса, которое указано в атрибуте name тега `<service>`.

##### *Цель*

Написать программу на Python, которая:

1. Получает на вход одну строку, содержащую данные конфигурации в формате XML (через стандартный ввод).
2. Парсит эту XML-строку.
3. Находит тег `<service>` (предполагается, что он находится непосредственно внутри корневого тега).
4. Извлекает из этого тега значение атрибута `name`.
5. Выводит на экран только значение этого атрибута.
