# Cybersecurity-Projects

Мини-проекты на Python по некоторым темам кибербезопасности (парсинг данных (текст, XML), регулярные выражения для анализа логов, криптография, парсинг и анализ логов безопасности, мониторинг системы и обнаружение аномалий), имеющие практическую направленность и симулирующие реальные сценарии работы специалистов по кибербезопасности.
