### **Анализ попыток входа: логи аутентификации**

##### *Описание задачи*

Файлы журналов аутентификации (auth.log в Linux) содержат важную информацию о попытках входа в систему. Анализ этих логов является ключевым элементом мониторинга безопасности, позволяя выявлять подозрительную активность, такую как множественные неудачные попытки входа (потенциальные атаки брутфорса) или несанкционированный доступ.

В этой задаче нам предстоит разработать скрипт для анализа файла auth.log, который подсчитывает успешные и неудачные попытки входа в систему и предоставляет детальную статистику по IP-адресам и пользователям.

##### *Цель*

Написать программу на Python, которая анализирует лог-файл аутентификации и предоставляет следующую информацию:

1. Общее количество успешных и неудачных попыток входа
2. Статистика по пользователям: какие пользователи успешно вошли, какие пользователи не смогли войти, сколько раз
3. Статистика по IP-адресам: с каких IP-адресов были успешные входы, с каких были неудачные, сколько раз

Формат строк лога, с которым мы будем работать, примерно такой:

May 21 12:34:56 hostname sshd[1234]: Accepted password for user1 from 192.168.1.100 port 22
May 21 12:35:22 hostname sshd[1235]: Failed password for user2 from 192.168.1.101 port 22
                 
Программа должна:

1. Считать данные из стандартного ввода
2. Проанализировать каждую строку, определяя успешные и неудачные входы
3. Извлечь имена пользователей и IP-адреса
4. Подсчитать статистику
5. Вывести результат в формате JSON

Ожидаемый формат вывода:

{
  "summary": {
    "successful_attempts": 4,
    "failed_attempts": 6,
    "total_attempts": 10
  },
  "users": {

    }
  },
  "ips": {

    }
  }
}
