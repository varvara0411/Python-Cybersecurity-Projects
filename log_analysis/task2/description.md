### **Охота на брутфорс: анализ логов SSH**

##### *Описание задачи*

SSH (Secure Shell) - это протокол, обеспечивающий безопасное удаленное управление серверами. Однако, именно из-за своей важности, SSH-серверы часто становятся мишенью для атак методом перебора паролей, известных как брутфорс (brute-force).

Атака брутфорса заключается в том, что злоумышленник пытается подобрать пароль к учетной записи, перебирая множество комбинаций. Характерным признаком такой атаки в логах аутентификации является большое количество неудачных попыток входа с одного IP-адреса за короткий промежуток времени.

##### *Цель*

Написать скрипт на Python, который анализирует лог-файл аутентификации (например, auth.log) и обнаруживает IP-адреса, с которых было совершено подозрительно много неудачных попыток входа по SSH. Это первый шаг к выявлению и блокированию потенциальных атак брутфорса.

Программа должна:

1. Считать данные лога из стандартного ввода
2. Найти все записи о неудачных попытках входа по SSH
3. Извлечь IP-адрес источника для каждой неудачной попытки
4. Подсчитать количество неудачных попыток для каждого IP-адреса
5. Определить IP-адреса, количество неудачных попыток с которых превышает 5 (заданный порог)
6. Вывести отсортированный список этих подозрительных IP-адресов, каждый на новой строке

Например, если с IP-адреса 192.168.1.100 было 6 неудачных попыток, а с 10.0.0.5 - только 2, то в выводе должен быть только IP-адрес 192.168.1.100.
